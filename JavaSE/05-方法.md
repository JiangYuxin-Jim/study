# 05 - Java 核心语法 4：方法

## 1. 方法概述

### 1.1 什么是方法？

方法就是一段**可重复调用**的代码块，用于完成特定功能。

```java
// 没有方法 → 代码重复
int a1 = 10, b1 = 20, sum1 = a1 + b1;
int a2 = 30, b2 = 40, sum2 = a2 + b2;
int a3 = 50, b3 = 60, sum3 = a3 + b3;

// 提取方法 → 一次定义，到处调用
public int add(int a, int b) {
    return a + b;
}
sum1 = add(10, 20);
sum2 = add(30, 40);
sum3 = add(50, 60);
```

### 1.2 方法的好处

| 好处 | 说明 |
|------|------|
| **复用** | 一次编写，多次调用 |
| **解耦** | 拆分复杂逻辑，每个方法职责单一 |
| **可维护** | 修改逻辑只改一处 |
| **可读性** | 方法名描述功能，代码像文档 |

## 2. 方法的定义与调用

### 2.1 完整格式

```java
修饰符 返回值类型 方法名(参数列表) {
    // 方法体
    return 返回值;   // 如果返回值类型是 void，可以省略
}
```

### 2.2 各部分说明

```java
public static int getMax(int a, int b) {
    //  ↑     ↑    ↑     ↑    ↑     ↑
    // 修饰符  静态  返回值 方法名  参数列表
    if (a > b) {
        return a;
    } else {
        return b;
    }
}
```

| 部分 | 说明 |
|------|------|
| **修饰符** | `public` `private` `protected` `static` 等 |
| **返回值类型** | 方法返回的数据类型，无返回值写 `void` |
| **方法名** | 小驼峰命名，见名知意 |
| **参数列表** | `(类型 参数名, 类型 参数名, ...)`，可以为空 |
| **方法体** | 具体逻辑 |
| **return** | 返回结果，`void` 中 `return` 可选（只用于提前结束） |

### 2.3 调用方式

```java
// 无返回值
printInfo();

// 有返回值
int result = getMax(10, 20);

// 返回值可以直接使用
System.out.println(getMax(10, 20));
int result2 = getMax(30, getMax(10, 20)); // 嵌套调用
```

## 3. 方法重载（Overload）

### 3.1 什么是重载？

**同一个类**中，方法名相同但**参数列表不同**，就叫方法重载。

```java
public class Calculator {
    // 重载 1：两个 int
    public int add(int a, int b) {
        return a + b;
    }

    // 重载 2：三个 int
    public int add(int a, int b, int c) {
        return a + b + c;
    }

    // 重载 3：两个 double
    public double add(double a, double b) {
        return a + b;
    }
}
```

### 3.2 重载的判断规则

**只看参数列表，不看返回值、不看参数名**：

```java
// ✅ 参数个数不同 — 是重载
int add(int a, int b)
int add(int a, int b, int c)

// ✅ 参数类型不同 — 是重载
int add(int a, int b)
double add(double a, double b)

// ✅ 参数顺序不同 — 是重载
void print(int a, String s)
void print(String s, int a)

// ❌ 只有返回值不同 — 不是重载（编译报错）
int add(int a, int b)
double add(int a, int b)       // 与参数无关，编译器无法区分

// ❌ 只有参数名不同 — 不是重载（编译报错）
int add(int a, int b)
int add(int x, int y)          // 类型和个数完全一样
```

### 3.3 println 的重载

```java
// System.out.println() 就是对所有基本类型做了重载
System.out.println(10);         // println(int)
System.out.println(3.14);       // println(double)
System.out.println('A');        // println(char)
System.out.println("hello");    // println(String)
System.out.println(true);       // println(boolean)
```

## 4. 参数传递

### 4.1 基本类型参数（值传递）

```java
public static void change(int x) {
    x = 100;   // 只改了局部变量 x，不影响外面的 a
}

public static void main(String[] args) {
    int a = 10;
    change(a);
    System.out.println(a);  // 输出 10（没有变！）
}
```

### 4.2 引用类型参数

```java
public static void change(int[] arr) {
    arr[0] = 100;  // 通过地址修改了堆内存中的数据
}

public static void main(String[] args) {
    int[] arr = {10, 20, 30};
    change(arr);
    System.out.println(arr[0]);  // 输出 100（变了！）
}
```

### 4.3 内存图解

```
基本类型（值传递）：               引用类型：
┌────────┐                       ┌────────┐
│ main   │                       │ main   │
│ a = 10 │                       │ arr ──┼──→ [0x100] ─→ {10,20,30}
└────────┘                       └────────┘
    │ copy 值                      │ copy 地址
    ▼                              ▼
┌────────┐                       ┌──────────────┐
│ change │                       │ change       │
│ x = 10 │ → x=100               │ arr → [0x100]│ → arr[0]=100
└────────┘  不影响 a              └──────────────┘   影响原数组！
```

> **一句话**：Java 中只有值传递。基本类型传的是数据值，引用类型传的是地址值。方法通过地址值可以修改堆内存中的数据。

## 5. 方法练习

### 5.1 数组遍历（格式化输出）

```java
public static void printArray(int[] arr) {
    System.out.print("[");
    for (int i = 0; i < arr.length; i++) {
        System.out.print(arr[i]);
        if (i < arr.length - 1) {
            System.out.print(", ");
        }
    }
    System.out.println("]");
}
// 输出：[10, 20, 30, 40, 50]
```

### 5.2 判断是否为素数

```java
public static boolean isPrime(int num) {
    if (num <= 1) return false;
    for (int i = 2; i <= Math.sqrt(num); i++) {
        if (num % i == 0) {
            return false;
        }
    }
    return true;
}
```

### 5.3 递归求阶乘

```java
public static int factorial(int n) {
    if (n == 1) {
        return 1;         // 递归出口
    }
    return n * factorial(n - 1);  // 递归调用
}
// factorial(5) = 5 * 4 * 3 * 2 * 1 = 120
```

> **递归注意**：必须有**递归出口**（终止条件），否则会栈溢出 `StackOverflowError`。

## 6. 总结

| 知识点 | 核心要点 |
|--------|----------|
| 方法定义 | `修饰符 返回值类型 方法名(参数列表) { 方法体 }` |
| void | 无返回值，`return;` 可选（仅用于提前结束） |
| 重载 | 方法名相同 + 参数列表不同，不管返回值 |
| 重载判断 | 看参数**个数/类型/顺序**，不是看**返回值/参数名** |
| 值传递 | 基本类型传数据值（不改原值），引用类型传地址（可改堆数据） |
| 方法的 `static` | 静态方法可直接调用，非静态方法需通过对象调用 |
| 递归 | 方法调用自己，必须有出口，否则栈溢出 |
| 方法设计原则 | 单一职责：一个方法只做一件事 |
