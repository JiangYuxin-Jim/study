```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐

│              │    │              │    │              │
│  语句1       │    │    条件?      │    │    条件?      │

│  语句2       │    │  true  false │    │  true  false │

│  语句3       │    │  代码A  代码B  │    │  循环体  退出  │
│              │    │              │    │   ↑_________| │
└──────────────┘    └──────────────┘    └──────────────┘
```

## 2. 分支结构

### 2.1 if 语句

```java
// 格式一：单分支
if (条件) {
    // 条件为 true 时执行
}

// 格式二：双分支
if (条件) {
    // 条件为 true 时执行
} else {
    // 条件为 false 时执行
}

// 格式三：多分支
if (条件1) {
    // 条件1为true
} else if (条件2) {
    // 条件2为true
} else if (条件3) {
    // 条件3为true
} else {
    // 以上都不满足
}
```

**注意**：`if` 后面只有一条语句时，`{}` 可以省略，但**不推荐**。

### 2.2 switch 语句

```java
switch (表达式) {                  // 表达式类型：byte/short/int/char/String/枚举
    case 值1:
        // 语句体1
        break;                    // 跳出 switch，没有 break 会穿透！
    case 值2:
        // 语句体2
        break;
    // ...
    default:                      // 所有 case 都不匹配时执行
        // 默认语句体
        break;
}
```

### 2.3 case 穿透

```java
int month = 3;
switch (month) {
    case 3:
    case 4:
    case 5:
        System.out.println("春季");    // month=3/4/5 都会走到这里
        break;
    case 6:
    case 7:
    case 8:
        System.out.println("夏季");
        break;
    // ...
}
```

> **穿透是故意利用**：多个 case 执行相同逻辑时，省略 break 让它们合并。

### 2.4 if vs switch

| | if | switch |
|------|----|--------|
| **判断类型** | 任意布尔条件（`>`, `<`, `!=` 等） | 等值匹配 |
| **适用场景** | 范围判断 | 值比较固定的情况 |
| **示例** | `if (score >= 90)` | `switch (dayOfWeek)` |

### 2.5 Java 14+ switch 新写法（了解）

```java
// 箭头语法 + 可以直接返回
String season = switch (month) {
    case 3, 4, 5 -> "春季";
    case 6, 7, 8 -> "夏季";
    case 9, 10, 11 -> "秋季";
    case 12, 1, 2 -> "冬季";
    default -> "无效月份";
};
```

## 3. 循环结构

### 3.1 for 循环

```java
// 格式
for (初始化语句; 条件判断; 步进语句) {
    // 循环体
}

// 示例：打印 1~10
for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}

// 执行顺序：
// 初始化 → 条件判断(true) → 循环体 → 步进 → 条件判断(true) → 循环体 → ... → 条件判断(false) → 结束
```

**常见写法**：

```java
// 遍历数组
for (int i = 0; i < arr.length; i++) { }

// 倒序
for (int i = 10; i >= 1; i--) { }

// 步长为 2（打印偶数）
for (int i = 0; i <= 100; i += 2) { }
```

### 3.2 while 循环

```java
// 格式
while (条件判断) {
    // 循环体
}

// 示例
int i = 1;
while (i <= 10) {
    System.out.println(i);
    i++;    // 步进语句写在循环体里，容易忘！
}
```

### 3.3 do-while 循环

```java
// 格式：至少执行一次循环体
do {
    // 循环体
} while (条件判断);

// 示例
int i = 11;
do {
    System.out.println("至少执行一次");   // 即使 i>10，也会打印一次
    i++;
} while (i <= 10);
```

### 3.4 三种循环对比

| | for | while | do-while |
|------|-----|-------|----------|
| **知道循环次数** | ✅ 首选 | ⚠️ | ⚠️ |
| **不知道循环次数** | ⚠️ | ✅ 首选 | ⚠️ |
| **至少执行一次** | ❌ | ❌ | ✅ 首选 |
| **变量作用域** | 循环体内 | 循环体外可继续用 | 同 while |

### 3.5 死循环（无限循环）

```java
// 三种写法
for (;;) {
    System.out.println("死循环");
}

while (true) {
    System.out.println("死循环");
}

do {
    System.out.println("死循环");
} while (true);
```

> 死循环内部必须有**跳出条件**（如 `break`），否则程序无法停止。

## 4. 跳转控制语句

### 4.1 break

```java
// 跳出当前循环（循环结束）
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;   // 循环到此为止，不再执行后续迭代
    }
    System.out.println(i);  // 输出 1, 2, 3, 4
}

// break 默认跳出最近一层循环
// 跳出外层循环：标签 break
outer:
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (j == 2) {
            break outer;    // 跳出外层循环
        }
    }
}
```

### 4.2 continue

```java
// 跳过本次，继续下一次
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        continue;  // 跳过 i=5，继续 i=6...
    }
    System.out.println(i);  // 输出 1~4, 6~10（缺 5）
}
```

### 4.3 break vs continue

```java
// break：整个循环终止
// continue：仅跳过当前这次迭代，循环继续

for (int i = 1; i <= 5; i++) {
    if (i == 3) break;      // 输出: 1 2
}

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;   // 输出: 1 2 4 5（跳过3）
}
```

## 5. 综合案例

### 5.1 打印 99 乘法表

```java
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        System.out.print(j + "×" + i + "=" + (i * j) + "\t");
    }
    System.out.println();  // 换行
}
```

### 5.2 判断素数

```java
int num = 17;
boolean isPrime = true;
for (int i = 2; i <= Math.sqrt(num); i++) {
    if (num % i == 0) {
        isPrime = false;
        break;              // 找到一个因子就够了
    }
}
System.out.println(isPrime ? "素数" : "不是素数");
```

## 6. 总结

| 知识点 | 核心要点 |
|--------|----------|
| 三种结构 | 顺序、分支、循环 |
| if 多分支 | `if` → `else if` → `else`，注意条件顺序（范围小的优先） |
| switch | 等值匹配，类型限制，`break` 防穿透 |
| case 穿透 | 故意不加 `break`，多个 case 合并处理 |
| for 循环 | 知道次数的首选：`for(初始化;条件;步进){}` |
| while 循环 | 不知道次数的首选：先判断再执行 |
| do-while | 至少执行一次 |
| break | 跳出当前循环（标签 break 可跳出外层） |
| continue | 跳过本次迭代，继续下一次 |
| 死循环 | `for(;;)` `while(true)`，内部必须可跳出 |
